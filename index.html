Интерактивный разбор сложных математических задач и удивительных парадоксов.

1. Линейная алгебра: Определители матриц

Пример 1: Матрица 2х2

Вычислить определитель матрицы А:

А =
35
24

Решение: Перемножаем крест-накрест элементы главной диагонали и вычитаем произведение побочной.

  1. Главная диагональ: 3 · 4 = 12.
  2. Побочная диагональ: 2 · 5 = 10.
  3. Разность результатов: 12 - 10 = 2.

Ответ: det(A) = 2

Пример 2: Матрица с отрицательными числами

Вычислить определитель матрицы B:

B =
1-3
42

Решение: Будьте внимательны со знаками минус при вычитании.

  1. Главная диагональ: 1 · 2 = 2.
  2. Побочная диагональ: 4 · (-3) = -12.
  3. Вычитаем отрицательное число (минус на минус дает плюс): 2 - (-12) = 2 + 12 = 14.

Ответ: det(B) = 14

2. Математический анализ: Производные и Пределы

Пример 1: Степенная функция и константа

Найти производную функции: у = 3x³ + 5x² - 7.

Решение: Применяем правило (xⁿ)' = n · xⁿ⁻¹ для каждого слагаемого.

  1. Первая часть: (3x³)' = 3 · 3x² = 9x².
  2. Вторая часть: (5x²)' = 5 · 2x = 10x.
  3. Число без икса зануляется: (7)' = 0.
y' = 9x² + 10x

Ответ: y' = 9x² + 10x

Пример 2: Производная с корнем и тригонометрией

Найти производную функции: y = √x + sin(x).

Решение: Используем табличные формулы для корня и синуса.

  1. Производная корня по таблице равна: (√x)' = 1 / (2√x).
  2. Производная синуса превращается в косинус: (sin(x))' = cos(x).
y' = 1 / (2√x) + cos(x)

Ответ: y' = 1 / (2√x) + cos(x)

Пример 3: Раскрытие неопределенности в пределе (lim)

Вычислить предел при x стремящемся к 3: lim (x² - 9) / (x - 3).

Решение: Если подставить x = 3, получится деление 0/0. Разложим числитель на множители.

  1. Разложим числитель по разности квадратов: (x² - 9) = (x - 3)(x + 3).
  2. Сокращаем одинаковую скобку (x - 3) в числителе и знаменателе.
  3. Остается простой предел: lim (x + 3). Подставляем x = 3: 3 + 3 = 6.
Ответ: 6

3. Интегралы и Дифференциальные уравнения

Пример 1: Многочлен

Найти интеграл: ∫(4x³ + 2x) dx.

Решение: Пользуемся правилом ∫xⁿ dx = (xⁿ⁺¹) / (n + 1).

  1. Первое слагаемое: ∫4x³ dx = 4 · (x⁴ / 4) = x⁴.
  2. Второе слагаемое: ∫2x dx = 2 · (x² / 2) = x².
  3. В конце обязательно дописываем букву + C.
∫(4x³ + 2x) dx = x⁴ + x² + C

Ответ: x⁴ + x² + C

Пример 2: Интеграл от экспоненты

Найти интеграл: ∫(eˣ + 3) dx.

Решение:

  1. Экспонента при интегрировании не меняется: ∫eˣ dx = eˣ.
  2. Интеграл от числа получает к себе в пару x: ∫3 dx = 3x.
∫(eˣ + 3) dx = eˣ + 3x + C

Ответ: eˣ + 3x + C

Пример 3: Дифференциальное уравнение 1-го порядка

Решить уравнение: dy / dx = 2x.

Решение: Это уравнение с разделяющимися переменными.

  1. Умножим обе части на dx: dy = 2x dx.
  2. Навесим знаки интеграла на обе стороны: ∫dy = ∫2x dx.
  3. Берем интегралы: левая превращается в y, правая — в x² + C.
y = x² + C

Ответ: y = x² + C

4. Комплексные числа и Векторы

Пример 1: Умножение комплексных чисел

Перемножить два числа: z₁ = 2 + 3i и z₂ = 1 - i.

Решение: Перемножаем скобки, учитывая, что i² = -1.

  1. Раскрываем скобки: (2 + 3i) · (1 - i) = 2 - 2i + 3i - 3i² = 2 + i - 3i².
  2. Заменяем i² на (-1): 2 + i - 3 · (-1) = 2 + i + 3.
  3. Складываем числа: 2 + 3 = 5.
Ответ: z = 5 + i

Пример 2: Скалярное произведение векторов

Найти скалярное произведение векторов a = {3; -1; 2} и b = {4; 2; -5}.

Решение: Попарно умножаем координаты X, Y, Z и складываем результаты.

  1. Умножаем координаты: (3 · 4) + ((-1) · 2) + (2 · (-5)).
  2. Считаем: 12 - 2 - 10 = 0.

Ответ: a · b = 0

5. Теория вероятностей

Пример 1: Классическое определение вероятности

В коробке лежат 5 синих, 3 красных и 2 зеленых шара. Какова вероятность вытащить наугад именно красный шар?

Решение: Вероятность — это отношение количества нужных исходов к общему числу исходов (P = m / n).

  1. Считаем общее число шаров: 5 + 3 + 2 = 10 (это n).
  2. Число подходящих (красных) шаров: 3 (это m).
  3. Находим вероятность по формуле P = 3 / 10.
P = 0.3 (или 30%)

Ответ: 0.3
